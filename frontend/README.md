# ScaleConfig Frontend

This is the React + TypeScript frontend for ScaleConfig.

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Lint and Test
```bash
npm run lint
npm test
```

## Environment
- `BASE_URL` (optional) sets the Vite base path for GitHub Pages.

## Scale Model Notes
The bundled samples include device strings for Budry MFD-51 TM-xA (V3.20E) and TM-xA UC123 V3.20E.
